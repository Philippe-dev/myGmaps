Édition d'un élément de carte

Titre
Le titre de l'élément de carte. Ce champ est obligatoire. Côté public, le titre s'affichera dans une fenêtre modale lors du clic sur l'élément, suivi éventuellement de sa description.
Type et position
Choisissez un type d'élément dans la barre d'outils, puis cliquez sur la carte pour le positionner. Pour insérer un autre type d'élément, il faut auparavant initialiser la carte, elle ne peut en effet contenir qu'un seul type d'élément à la fois..
Action du clic sur les éléments
  • si une icône est affichée, un clic permet d'en changer en choisissant parmi les icônes par défaut ou vos icônes personnelles.
  • si une ligne brisée, un polygone, un rectangle ou un cercle sont affichés, le clic permet de choisir la couleur du trait, celle de remplissage, ainsi que l'épaisseur du trait.
  • si une ligne brisée ou un polygone sont affichés, le clic droit sur un point d'ancrage permet de le supprimer.
  • si un fichier kml ou un flux GeoRSS sont affichés, le clic permet de modifier la source de l'élément.

Côté public, un clic sur un élément affichera son titre et sa description dans une fenêtre modale, sauf pour les fichiers Kml, Kmz ou GeoRSS qui gèrent directement leurs descriptions.

Description
La description du point d'intérêt, de la ligne brisée, du polygone, du rectangle ou du cercle contenus dans la carte. Côté public, cette description s'affichera dans une fenêtre modale lors du clic sur l'élément. Si ce champ est laissé vide, seul le titre de l'élément s'affichera dans la fenêtre modale. Un fichier Kml ou GéoRSS inclus ne peut pas avoir de description, c'est en effet le fichier qui fournit lui-même le contenu des fenêtres modales.
Notes
Cette zone de texte sert à la prise de notes ou pense-bête divers. Elle ne sera jamais affichée.
État de l'élément de carte
Permet de choisir l'état de l'élément de carte après enregistrement :
  • En attente : l'élément de carte est en attente de publication. Il ne peut pas être inséré dans un billet
  • Programmé : l'élément de carte sera mis en ligne aux date et heure indiquées dans le champ Publié le.
  • Non publié : élément de carte hors ligne.
  • Publié : l'élément de carte est disponible pour l'ajout dans une carte.
Publié le
Permet de modifier la date et l'heure de publication de l'élément de carte.
Langue de l'élément
Code langue de votre élément. Par défaut, il s'agit du code de votre langue. Vous pouvez le changer, par le code d'une autre langue, par exemple "en", "fr-qc". Ce code est libre, il sera utilisé lors de l'affichage de l'élément.
Format du texte
Permet de choisir la syntaxe de saisie utilisée pour la description de l'élément de carte. Le wiki est une syntaxe simplifiée et sera converti en xhtml valide ; à moins que vous maîtrisiez parfaitement le HTML nous vous conseillons de la choisir de préférence. Consultez la référence de la syntaxe Wiki pour plus d'informations.
Élement sélectionné
Les éléments marqués comme sélectionnés peuvent être retrouvés plus simplement avec un filtre dans la liste des éléments.
Catégorie
La catégorie de votre élément de carte. Vous pouvez créer une nouvelle catégorie, ou n'affecter votre élément de carte à aucune catégorie en choisissant la ligne vide.

Types d'élements de cartes

Choisissez dans la barre d'outils le type d'élément que vous voulez ajouter puis cliquez sur la carte. Un élément de carte peut avoir les types suivants :

Type non défini
Aucun marqueur n'est présent sur la carte, cet élément ne peut être utilisé et doit être considéré comme un brouillon.
Point d'intérêt
Une icône est affichée sur la carte.
Ligne brisée
Plusieurs points sont présents sur la carte, reliés par une ligne polygonale.
Polygone
Plusieurs points sont présents sur la carte, reliés par une ligne fermée matérialisant un polygone.
Rectangle
Un rectangle est affiché sur la carte.
Cercle
Un cercle est affiché sur la carte.
Fichier kml inclus
La carte affiche des objets issus d'un fichier kml. Ils ne sont toutefois pas modifiables. Ce type d'élément ne peut avoir de description..
Flux GeoRSS
La carte affiche des objets issus d'un flux GeoRSS. Ils ne sont toutefois pas modifiables. Ce type d'élément ne peut avoir de description.
Itinéraire
La carte affiche un itinéraire routier entre deux points et un panneau latéral comportant des indications de conduite. Vous ne pouvez utiliser qu'un seul itinéraire sur une même carte.

Style des cartes

Les cartes peuvent être stylées au moyen de fichiers de style. À titre d'exemple, un fichier de style est inclus dans le plugin, il se nomme Neutral Blue et apparaît à côté des boutons habituels Plan et Satellite.

Vous pouvez ajouter des styles personnels provenant de votre médiathèque, qui apparaîtront alors sur cette ligne de boutons. Voici comment procéder :

Emplacement des fichiers de style
  • pour ajouter des styles, créez un dossier nommé myGmaps dans votre médiathèque
  • dans ce dossier, créez un autre dossier nommé styles
  • ranger dans ce dossier vos styles personnels
Nom et contenu des fichiers de style
  • le nom d'un fichier de style doit correspondre au masque suivant : mon_style_styles.js où mon_style est le nom que vous pouvez modifier (ce nom n'est qu'un simple identifiant et n'apparaîtra pas en partie publique).
  • le contenu du fichier doit contenir un tableau JSON de type : 
    [{featureType: '',elementType: '',stylers: [{hue: ''},{saturation: ''},{lightness: ''},// etc...]},{featureType: '',// etc...}]

Vous pouvez obtenir de nombreux styles pour vos cartes sur le site Snazzy Maps.

Icônes et fichiers kml personnels

Icônes personnelles
Par défaut, seules quelques icônes sont fournies, mais il est très simple d'ajouter les vôtres :
  • dans votre médiathèque, créez un dossier nommé myGmaps
  • dans le dossier myGmaps créez un autre dossier nommé icons
  • ranger dans ce dossier vos icônes personnelles.
Fichiers Kml personnels
Vous pouvez inclure un fichier Kml en fournissant son URL, ou utiliser un fichier de votre médiathèque. Il est très simple d'ajouter les vôtres :
  • dans votre médiathèque, créez un dossier nommé myGmaps
  • dans le dossier myGmaps créez un autre dossier nommé kml_files
  • ranger dans ce dossier vos fichiers kml personnels.
